Clifford es una herramienta para completar instancias de formularios electrónicos.

Los pasos a seguir son los siguientes:

  • Seleccionar la agrupación de formularios find_replace.
  • Acceder a la relación de formularios disponibles para la agrupación seleccionada web.
  • Ver la relación de instancias del formulario seleccionado topic.
  • Crear una nueva instancia add. La agrupación de formularios puede estar configurada para permitir una única instancia, en cuyo caso el icono no volverá a aparecer, salvo si se borra la instancia.
  • La instancia solo se guarda en la base de datos si se cumplen todas las validaciones. No se puede almacenar un borrador.
  • Una vez almacenada la instancia en la base de datos se recibirá un mensaje de confirmación y se verá en la relación de instancias con la fecha y hora en que se ha creado.
  • Ver una instancia visibility.
  • Modificar una instancia create.
  • Se puede abandonar la edición haciendo click en cualquier parte fuera de la ventana de edición.
  • Una vez almacenada la instancia en la base de datos se recibirá un mensaje de confirmación y se verá la fecha y hora de modificación actualizada.
  • Borrar una instancia delete_forever.
  • Obtener un volcado de pantalla de una instancia en PDF description.

Las opciones son configurables por el administrador por lo que no siempre aparecen.
